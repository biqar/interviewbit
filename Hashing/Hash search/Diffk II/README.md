# Diffk II

Asked in: [Facebook](#)

## Problem Statement
Given an array `A` of integers and another non negative integer `k`, find if there exists 2 indices `i` and `j` such that `A[i] - A[j] = k`, `i != j`.

Example :

Input :
```
A : [1 5 3]
k : 2
```

Output :
```
1
```
as `3 - 1 = 2`

Return `0` / `1` for this problem.
